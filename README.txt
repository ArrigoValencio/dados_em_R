
Este repositório traz material oriundo de estudos estatísticos utilizando o software livre R.

O arquivo probability_and_data_project é o projeto final de conclusão do curso online Introduction
to Probability and Data with R, oferecido pela Duke University através da plataforma Coursera.
Especificamente, nesse projeto são respondidas três perguntas de pesquisa utilizando dados do
BRFSS (Behavioral Risk Factor Surveillance System) que é um projeto colaborativo de vigilância 
de fatores de risco comportamental e que coleta dados da população nos EUA. Respondem-se essas
perguntas de pesquisa no intuito de exercitar e demonstrar o aprendizado com o curso, o que rendeu
um certificado de conclusão. Esse projeto está redigido em língua inglesa.
